## Documentation

#### Installation:

Add the package to Sublime Text 3 packages directory:

```zsh
$ ./bin/download.sh
```

Or create a symlink to snippets package

```
$ ./bin/symlink.sh
```

#### MySQL Snippets:

- `clivern-mysql-db-create`: Create Database.
- `clivern-mysql-db-drop`: Drop Database.
